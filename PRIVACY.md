Política de Privacidade – Prosa

Última atualização: 21 de junho de 2026

1. Introdução

O Prosa é um editor de texto desktop, open source, desenvolvido pela Rodrigo Brito, projetado para criar, editar, importar e exportar documentos de texto de forma simples, moderna e segura.

Esta Política de Privacidade explica como o Prosa trata as informações dos usuários.

2. Coleta de dados

O Prosa não coleta, armazena, transmite ou compartilha dados pessoais dos usuários.

O aplicativo funciona localmente no computador do usuário e não requer:

criação de contas;
cadastro de usuários;
login;
autenticação em serviços externos;
armazenamento em nuvem;
sincronização online.
3. Arquivos e documentos

Os documentos abertos, criados ou salvos pelo Prosa permanecem sob controle exclusivo do usuário.

O aplicativo acessa arquivos locais apenas quando o usuário realiza ações explícitas, como:

abrir um documento;
salvar um documento;
importar arquivos;
exportar arquivos;
inserir imagens.

Nenhum documento é enviado para servidores externos.

4. Permissões utilizadas

O Prosa pode acessar recursos do sistema operacional exclusivamente para fornecer suas funcionalidades, incluindo:

sistema de arquivos;
impressoras;
área de transferência (copiar e colar);
seleção de imagens;
fontes instaladas no computador.

Esses acessos ocorrem somente mediante ação direta do usuário.

5. Conexão com a internet

O Prosa não depende de conexão com a internet para funcionar.

O aplicativo não transmite dados de uso, documentos ou informações pessoais para servidores próprios ou de terceiros.

Eventuais conexões de rede podem ocorrer apenas para:

verificação manual de atualizações (quando implementada);
acesso a links externos disponibilizados pelo usuário.
6. Telemetria e análise de uso

O Prosa não utiliza ferramentas de telemetria, rastreamento ou análise de comportamento, incluindo, mas não se limitando a:

Google Analytics;
Microsoft Clarity;
Sentry;
Mixpanel;
ferramentas de publicidade.
7. Cookies

O Prosa não utiliza cookies.

8. Compartilhamento de informações

Nenhuma informação é compartilhada com terceiros.

9. Segurança

Como o processamento dos documentos ocorre localmente, a segurança dos arquivos depende principalmente das práticas adotadas pelo próprio usuário e pelo sistema operacional utilizado.

Recomenda-se:

manter o sistema operacional atualizado;
utilizar senhas de acesso ao computador;
realizar backups periódicos dos documentos.
10. Software livre

O Prosa é distribuído como software livre sob a licença GNU General Public License v3.0 (GPL-3.0-or-later). O código-fonte está disponível publicamente para auditoria e contribuição.

11. Alterações nesta política

Esta Política de Privacidade poderá ser atualizada futuramente para refletir novas funcionalidades.

A data da última atualização será sempre informada no início deste documento.
